Exact Guides

Version: 1.0 | Developer: 3Draco

1. View Options

  • Show Guides: Toggles the visibility of all guides on your canvas.
  • Show Rulers: Shows or hides the Krita rulers at the top and left edges.
  • Lock Guides: Prevents guides from being moved accidentally with the mouse.
  • Snap: Enables snapping, so your brush or selections "stick" to the guides.

2. Layout Management

You can save your current guide configuration for future use:

  • Save: Give your current layout a name to store it on your hard drive.
  • Load: Choose a previously saved layout from the dropdown menu and apply it to your current document.
  • Delete: Removes the selected layout from your saved list.

3. Units: Pixels vs. Percentage

This tool allows you to work with two different units:

  • Pixels (px): Ideal for technical drawings or web design where fixed sizes are required.
  • Percentage (%): Perfect for artistic compositions (like the Rule of Thirds). Percentage guides remain relative to the image size, even if you change the unit back and forth.

4. Custom Names and Direct Entry

In the table, you can see a list of all active guides:

  • Renaming: Double-click the name (e.g., "H-0") to give it a custom description (e.g., "Horizon").
  • Direct Entry: Double-click the position value to type in an exact coordinate. In percentage mode, you can use decimals (e.g., 33.33).

5. Update and Maintenance

  • Update (Refresh): Since Krita does not always notify plugins when a guide is moved manually with the mouse, use this button to synchronize the list with the canvas.
  • Clear All: Deletes all horizontal and vertical guides from the current document at once.

6. Image Split

This button opens Krita's built-in "Image Split" tool, allowing you to quickly divide your canvas into equal sections or prepare it for export.

Exact Guides (Deutsch)

Version: 1.0 | Ersteller: 3Draco

1. Ansichts-Optionen

  • Guides anzeigen: Schaltet die Sichtbarkeit aller Hilfslinien auf deiner Leinwand um.
  • Lineale anzeigen: Blendet die Krita-Lineale am oberen und linken Rand ein oder aus.
  • Guides sperren: Verhindert, dass Hilfslinien versehentlich mit der Maus verschoben werden.
  • Einrasten: Aktiviert das "Snap"-Verhalten, sodass Pinsel oder Auswahlen an den Linien haften bleiben.

2. Layout-Verwaltung

Du kannst deine aktuelle Konfiguration von Hilfslinien für die zukünftige Verwendung speichern:

  • Speichern: Gib deinem aktuellen Layout einen Namen, um es dauerhaft auf der Festplatte zu speichern.
  • Laden: Wähle ein zuvor gespeichertes Layout aus dem Menü aus, um es auf das aktuelle Dokument anzuwenden.
  • Löschen: Entfernt das ausgewählte Layout aus deiner Liste.

3. Einheiten: Pixel vs. Prozent

Dieses Tool ermöglicht das Arbeiten in zwei verschiedenen Einheiten:

  • Pixel (px): Ideal für technisches Zeichnen oder Webdesign, wo feste Größen wichtig sind.
  • Prozent (%): Perfekt für künstlerische Kompositionen (z. B. Drittel-Regel). Prozentuale Linien bleiben relativ zur Bildgröße erhalten.

4. Eigene Namen und Direkteingabe

In der Tabelle siehst du alle aktiven Hilfslinien:

  • Benennen: Klicke doppelt auf den Namen (z. B. "H-0"), um eine eigene Beschreibung einzugeben (z. B. "Horizont").
  • Direkteingabe: Klicke doppelt auf den Positionswert, um eine exakte Koordinate einzutippen. Im Prozentmodus werden Dezimalstellen (z. B. 33.33) unterstützt.

5. Aktualisierung und Wartung

  • Aktualisieren: Da Krita Plugins nicht immer automatisch benachrichtigt, wenn eine Linie manuell verschoben wird, nutzt du diesen Button, um die Liste zu synchronisieren.
  • Alle löschen: Entfernt sofort alle horizontalen und vertikalen Hilfslinien aus dem aktuellen Dokument.

6. Bild unterteilen

Dieser Button öffnet Krita's integriertes "Bild unterteilen"-Werkzeug, mit dem du die Leinwand schnell in gleichgroße Abschnitte zerlegen kannst.
